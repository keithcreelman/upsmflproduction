# Rulebook Source Archive

Place source rulebook documents and text extracts in this folder when rebuilding `data/rules.json`.

The builder reads these files by default:
- `current_rulebook_struct.json`
- `UPS Rule Book.txt`
- `UPS Contract Rules.txt`
- `UPS FFL Bylaws V2014.1.txt`
- `UPS Dynasty Cap Rulebook 2013.1.txt`

Override source location with:
- `RULEBOOK_ARCHIVE_DIR=/absolute/path/to/archive`
